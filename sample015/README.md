WordPress ワードプレスでページの種類ごとの処理をしたい場合の条件分岐の関数 20250403

1. 条件分岐の概要
WordPressでは、特定のページや条件に応じて処理を分岐させることができます。
例えば、トップページだけに特定のコンテンツを表示したり、特定の固定ページでのみスクリプトを読み込むことが可能です。


2. トップページの条件分岐
トップページだけで処理を行うには、以下のようなコードを使用します：

; 「固定ページ」かどうか？ &&(かつ) 「投稿ページ」かどうか？

if (is_front_page() && is_home()) {
    // トップページでのみ実行する処理
}

is_front_page(): トップページを判定。
is_home():  を判定。
両方を組み合わせることで、トップページのみの処理を実現します。

is_front_page():
WordPressの関数です。「フロントページ」であるかどうかを判定します。
フロントページとは、WordPressの「表示設定」で設定されている「ホームページの表示」のオプションで 固定ページ を選択している場合を指します。




front-page.php
役割:
WordPressの「ホームページの表示」で 「固定ページ」 をフロントページとして設定した場合に使用されるテンプレートファイルです。
サイトのトップページ（フロントページ）をカスタマイズするために使用されます。

優先順位:
テーマに front-page.php が存在する場合、最も優先されます。
これは、トップページ専用のテンプレートとして動作します。

使われるケース:
トップページを他のページとは完全に異なるデザインやレイアウトにしたい場合。


is_home():
WordPressの関数です。「投稿ページ」であるかどうかを判定します。
投稿ページとは、WordPressの「表示設定」で設定されている「ホームページの表示」が「最新の投稿」に設定されている場合のページです。


home.php
役割:
WordPressの「ホームページの表示」で 「最新の投稿」 を表示する場合に使用されるテンプレートファイルです。
ブログの投稿一覧ページのテンプレートとしても機能します。

優先順位:
front-page.php がない場合に使われます。
ブログ一覧ページとして特化させたいときに使用されることが多いです。

使われるケース:
投稿一覧ページのデザインをカスタマイズしたい場合。


優先順位の違い
以下は、WordPressがトップページ用のテンプレートを選ぶ際の優先順位です：

front-page.php:
「固定ページ」をフロントページとして設定した場合、最優先で使用されます。

home.php:
「最新の投稿」をフロントページとして設定した場合に使われます。
また、ブログ投稿一覧用のテンプレートとしても動作します。

その他のテンプレート（index.php など）:
front-page.php や home.php が存在しない場合にフォールバックとして使用されます。

まとめ
front-page.php: トップページ専用。サイト全体の最初の印象をデザインするために最適。
home.php: 投稿一覧用。ブログ記事が中心のサイト向け。



3. 固定ページの条件分岐
固定ページでのみ処理を行うには、以下のコードを使用します：
if (is_page()) {
    // 固定ページでのみ実行する処理
}


特定の固定ページを指定する場合：
if (is_page('contact')) {
    // 固定ページ「contact」でのみ実行する処理
}
